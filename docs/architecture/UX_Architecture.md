## Section 7 вЂ” UX Principles

1. Upload should always be one click away.

2. AI should explain every conclusion.

3. Important information appears before detailed information.

4. Never overwhelm the user.

5. Every page should have one primary action.

6. Loading should always show progress.

7. Errors should explain how to recover.

8. AI responses should reference the document.

9. Users should never lose uploaded work.

10. Every workflow should be completable within three minutes.
